# Köz Urfa — Maquette site web

Maquette de site vitrine (one-page) pour **Köz Urfa Ocakbaşı**, restaurant turc de grillades au feu de bois à Argenteuil.

- 📍 81 rue Paul Vaillant Couturier, 95100 Argenteuil
- 📞 09 86 76 87 09
- 📷 Instagram : [@kozurfa60](https://instagram.com/kozurfa60)

## Aperçu

Ouvrir simplement `index.html` dans un navigateur (double-clic), ou via GitHub Pages.

## Sections

Hero · Le restaurant · La carte (onglets) · Galerie · Avis · Infos & accès · Réservation.

## Statut

> ⚠️ Maquette de démonstration commerciale. Coordonnées réelles (source Google).
> Les **prix, horaires détaillés et photos** restent à valider avec le client.

## Stack

HTML + CSS + un peu de JavaScript (onglets de la carte). Aucun framework, aucune dépendance.
